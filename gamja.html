rx - how to gamja

gamja is a fairly popular and FOSS IRC webclient. It is a good choice for IRC newbies. As such, we host (a fork of) it at https://gamja.irc.andrewyu.org/.

Connecting

When launching gamja you'll be met with the following form:

Connect to rx

Nickname:
[______________________________________]
Password:            
[______________________________________]

[X] Auto-join channels #support, #chat
[ ] Remember me

 >  Advanced options

[ Connect ]

The password is only required if you registered your nickname. Just enter a nickname, click on "Connect" and start chatting!

Embedding

You can either embed a client into your web page, or craft a URL others can use to join your channel. For example, the following automatically joins #support:

https://gamja.irc.andrewyu.org/#support

The following automatically joins #support and sets the nickname to "IRCUser":

https://gamja.irc.andrewyu.org/?nick=IRCUser#support

You can embed gamja like so:

<iframe src="https://web.libera.chat/gamja/#libera" style="border:0; width:100%; height:450px;"></iframe>

Need help?

You may run the following:

/HELP

Alternatively, we're available in the #support channel for any questions that may arise.
